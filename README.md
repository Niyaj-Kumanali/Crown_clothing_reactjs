# 👑 Crown Clothing E-commerce

A modern, high-performance e-commerce platform built with **React**, **Vite**, **Redux Toolkit**, and **Firebase**. This project features a clean, professional UI designed with **Material UI** and is optimized for production.

## 🚀 Live Demo
You can view the live application here:
[**crown-clothing-fashion.vercel.app**](https://crown-clothing-fashion.vercel.app)

---

## ✨ Features
- **Modern Tech Stack**: Migrated from CRA to **Vite** for ⚡ lightning-fast development.
- **Advanced State Management**: Powered by **Redux Toolkit (RTK)** for predictable and efficient state handling.
- **Premium UI/UX**: Rebuilt with **Material UI** following modern design principles.
- **Fast & Responsive**: Optimized with **React 18**, lazy loading, and code splitting.
- **Secure Authentication**: Integrated with **Firebase Auth** (Google & Email/Password).
- **Persistence**: Cart and session persistence using **Redux Persist**.

## 🛠️ Getting Started

### 1. Installation
Clone the repository and install dependencies:
```bash
npm install
```

### 2. Environment Variables
Create a `.env` file in the root directory and add your Firebase credentials:
```env
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_project.appspot.com
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
```

### 3. Development
Run the development server:
```bash
npm run dev
```
